==================== PROGRAM HIMPUNAN ====================
Program himpunan memiliki 3 himpunan yaitu, himpunan POLBAN sebagai superset, himpunan UKM-1 sebagai subset-1, dan himpunan UKM-2 sebagai subset-2.
Dalam konteks ini, setiap anggota UKM-1 dan UKM-2 juga merupakan anggota dari POLBAN.

TUJUAN
Tujuan dari program himpunan ini adalah untuk memodelkan hubungan antara himpunan dalam konteks keanggotaan organisasi suatu universitas.
Dengan menggunakan himpunan sebagai model, dapat dilakukan berbagai operasi dan analisis terhadap anggota organisasi universitas.

CARA MENGGUNAKAN
1. Jalankan program himpunan pada IDE masing-masing.
2. Setelah program berhasil dijalankan, maka pengguna akan ditampilkan dengan display menu.
3. Dalam display menu, pengguna dapat melakukan beberapa fitur, diantaranya :
	 [1] Buat Himpunan
	     Pada menu ini pengguna dapat menambahkan anggota mahasiswa POLBAN sebagai
	     superset, dan juga dapat menambahkan anggota pada UKM POLBAN sebagai subset.
	 [2] Operasi Himpunan
	     Pada operasi himpunan, pengguna dapat menampilkan operasi himpunan seperti
	     irisan, gabungan, selisih, dan komplemen sesuai kebutuhan.
	 [3] Edit Daftar Himpunan
	     Pada menu ini, pengguna dapat mengubah atau menghapus anggota superset maupun anggota subset
	 [4] Cari Anggota Himpunan
	     Pada menu ini pengguna dapat memasukkan nama dan NIM dari mahasiswa,
	     kemudian program akan menampilkan mahasiswa tergabung dalam UKM apa saja.
	 [5] Tampilkan Daftar Himpunan
	     Pada operasi himpunan, pengguna dapat memilih untuk melihat anggota himpunan
	     secara Log, CURRENT, dan menampilkan GUIDE atau panduan penggunaan aplikasi.
